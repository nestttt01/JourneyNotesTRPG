遊戲玩法與系統指南

第一次玩，從這裡開始

完成 API 驗證並選好模型後，建立角色與情境、進入存檔，輸入第一個行動就能開始。（金鑰設定與錯誤排除看首頁說明。）

建立你的故事：角色、情境與規則

在「劇本創角」裡設定角色、NPC、情境與遊戲選項；情境也可交給 AI 隨機生成。

核心準則：寫進去的鐵則 AI 每回合優先讀取、不會違反。適合固定世界觀、人物設定，或鎖定場景（例如寫「B 目前不在此情境」擋掉不在場的 NPC）。

本場目標：替情境設一個目標，DM 會自然朝它推進，達成會觸發章節收束；留空則自由發展。

怎麼玩：行動、選項與模式

在輸入框打角色想說或想做的事，或點 AI 給的選項；有風險的行動用骰子按鈕做 D20 判定。

想安排環境、鏡頭或讓 NPC 行動，用 ［切換旁白］／［旁白］；切換回玩家用 ［切換玩家］／［玩家］；想直接改場景、時間、人物，用「神」按鈕。這些都不會被當成玩家發言。

已送出的文字要修改、複製或收藏，桌機右鍵、手機長按開選單。

生存與危機：HP／SAN

HP 或 SAN 偏低會進入危機：檢定變難，畫面也會出現暗角、雜訊等干擾，越接近歸零越強烈。

覺得干擾太多，可在「顯示與輸出設定 → 低狀態干擾」調成減弱或關閉。

SAN 很低時 AI 選項可能被污染：點下去有機率變成隱藏的失控行動並直接送出。

難度、死亡與收束

三種難度：標準歸零不會結束；困難歸零要擲生死檢定保命，NPC 只有一次復活；極限最難，NPC 死亡無法復活。

極限模式歸零會觸發「最後掙扎」，擲生死檢定通過就撐住，並拿到幾回合護盾。

「喘息」鈕可主動擲檢定回一點 HP／SAN，用來脫離死亡螺旋（回復上限為 HP／SAN 總量的 50%，不能連按）。

部分劇情道具有回復效果，在角色面板道具區按「使用」。

日常劇情也能回復：充分休息、過夜、接受治療或被安撫時，AI 會依事件小幅回復 HP／SAN（重複躺著刷不了血）。

名稱帶屬性標記的裝備道具（例如「幸運硬幣（智力+1）」）持有期間會自動加進對應骰點，失去道具即失效。

屬性點滿 20 後，該屬性按鈕會轉為鍛造：花 2 成長點兌換「成長結晶」（該屬性判定 +1，與裝備共用 +3 上限）；結晶是道具，可能因劇情失去。

成長與里程碑

達成重大成就會拿到成長點，可在「角色成長」區提升屬性或新增擅長領域，稱號隨成就升階。

達成情境的「本場目標」會觸發章節收束，由 AI 收尾並給成長點。

NPC 好感滿值、關係破裂或死亡等重大節點，系統會自動記進 Flags 與冒險日誌；好感跌到谷底（−100）會解鎖特殊成就。

面板上的像素愛心會隨好感改變動作：越愛越活潑，滿值會發光跳躍，觸底則整顆倒下。

冒險日誌記完整事件；想收藏對話用日記（泡泡右鍵、長按選「收藏這段」），在日記頁面對翻頁列桌機右鍵、手機長按可把整篇存成長圖。

讓 AI 不忘記、不跳針

故事變長後 AI 可能變鈍，這是模型限制。可以換標有 ▪ 的模型、多用「整理摘要／整理紀錄」壓縮記憶，或開「神」模式直接說明目前場景與重點。

存檔、備份與成人模式

每份遊戲紀錄綁定一份配置；另存新檔會複製紀錄並建立專屬配置。

匯出備份會保存紀錄、配置與頭像（不含 API Key），方便換裝置或還原。

成人模式在狀態面板開關（預設關），開啟後 AI 會在角色皆成年且合意的前提下更自然承接親密場景，實際尺度仍取決於所選模型。

API 金鑰與模型指南

開始前先知道這些

本遊戲需要使用你自己的 API Key 才能呼叫 AI。請先在首頁選擇供應商、貼上對應金鑰並完成驗證；金鑰不會包含在遊戲備份中，共用裝置也不建議開啟保存功能。

1. API Key 是什麼？

API Key 是 AI 服務用來辨識使用者與計算用量的金鑰，不是遊戲帳號或遊戲密碼。請向 Google AI Studio、OpenRouter 或 Anthropic Console 申請，並妥善保管；不要把金鑰貼到公開貼文、截圖或傳給其他人。

2. Google Gemini、OpenRouter 與 Anthropic

選擇 Google Gemini 用 Google AI Studio 的金鑰；選擇 OpenRouter 用 OpenRouter 金鑰（可跨多家模型）；選擇 Anthropic（Claude）用 Anthropic 官方金鑰（直接使用 Claude 模型）。三種金鑰不能互換，可用模型清單會在驗證後載入。

3. 模型怎麼選？

輕量模型通常回覆較快、用量較省，適合測試設定或日常遊玩；較大型模型通常更擅長長篇劇情、人物語氣與複雜指令。清單中標 ▪ 的是大廠通用優質模型，會排在前面方便挑選。模型表現與費用可能不同，請以供應商顯示的資訊為準。

4. 金鑰會保存在哪裡？

未開啟「在這台裝置保留金鑰」時，金鑰只在目前頁面使用，重新開啟後需要再次輸入；開啟後才會保存在這台裝置的瀏覽器。匯出備份不包含 API Key，因此在新裝置匯入遊戲資料後仍需重新驗證。

5. 哪些資料會傳給 AI？

為了產生回覆，角色設定、目前情境、必要的劇情摘要與近期對話會送往所選供應商。角色照片與完整本機存檔不會隨一般對話請求上傳；照片與遊戲進度主要保存在目前裝置。

6. 驗證或回覆失敗怎麼辦？

請依序確認供應商是否選對、金鑰是否完整、帳戶額度與模型權限是否有效，以及網路是否正常。若只有特定模型失敗，可以改選其他模型；等待過久時可按「取消等待」，未完成的輸入會盡量放回輸入框。

API 說明

這個遊戲會用你的 API Key 呼叫 AI。請先選擇服務商、貼上對應金鑰並完成驗證；金鑰不會被放進遊戲備份。

1. API 金鑰

API Key 是 AI 服務用來辨識使用者與計算用量的金鑰，不是遊戲帳號或遊戲密碼。請向 Google AI Studio、OpenRouter 或 Anthropic Console 申請，並妥善保管。

2. Gemini、OpenRouter 與 Anthropic

選擇 Google Gemini 用 Google AI Studio 的金鑰；選擇 OpenRouter 用 OpenRouter 金鑰（可跨多家模型）；選擇 Anthropic（Claude）用 Anthropic 官方金鑰（直接使用 Claude 模型）。三種金鑰不能互換，可用模型清單會在驗證後載入。

3. 模型怎麼選？

輕量模型通常回覆較快、用量較省，適合測試設定或日常遊玩；較大型模型通常更擅長長篇劇情、人物語氣與複雜指令。

4. 金鑰保存

未開啟「在這台裝置保留金鑰」時，金鑰只在目前頁面使用。共用裝置請勿開啟保存功能；匯出備份不包含 API Key。

5. 驗證失敗時

請先確認服務商與金鑰來源一致，再檢查金鑰是否完整貼上、是否有多餘空白、帳號是否已開通 API 權限。模型清單會依帳號與服務狀態變動。

6. 使用量與隱私

遊戲會把劇情上下文送到你選擇的 AI 服務商，用量與費用由該服務商計算。匯出遊戲備份不會包含 API Key；換裝置後需要重新驗證。

7. 模型建議

驗證後的模型清單會把大廠通用優質模型（Gemini、Claude、GPT、DeepSeek 等）標上 ▪ 並排在前面，這些模型穩定、擅長長篇與複雜指令。測試設定可先用較快、較省的；長篇或複雜劇情建議選標 ▪ 的模型。模型可在遊戲內隨時切換。

第一次玩，從這裡開始

完成 API 驗證並選好模型後，建立角色與情境、進入存檔，輸入第一個行動就能開始。（金鑰設定與錯誤排除看首頁說明。）

建立你的故事：角色、情境與規則

在「劇本創角」裡設定角色、NPC、情境與遊戲選項；情境也可交給 AI 隨機生成。

核心準則：寫進去的鐵則 AI 每回合優先讀取、不會違反。適合固定世界觀、人物設定，或鎖定場景（例如寫「B 目前不在此情境」擋掉不在場的 NPC）。

本場目標：替情境設一個目標，DM 會自然朝它推進，達成會觸發章節收束；留空則自由發展。

怎麼玩：行動、選項與模式

在輸入框打角色想說或想做的事，或點 AI 給的選項；有風險的行動用骰子按鈕做 D20 判定。

想安排環境、鏡頭或讓 NPC 行動，用 ［切換旁白］／［旁白］；切換回玩家用 ［切換玩家］／［玩家］；想直接改場景、時間、人物，用「神」按鈕。這些都不會被當成玩家發言。

已送出的文字要修改、複製或收藏，桌機右鍵、手機長按開選單。

生存與危機：HP／SAN

HP 或 SAN 偏低會進入危機：檢定變難，畫面也會出現暗角、雜訊等干擾，越接近歸零越強烈。

覺得干擾太多，可在「顯示與輸出設定 → 低狀態干擾」調成減弱或關閉。

SAN 很低時 AI 選項可能被污染：點下去有機率變成隱藏的失控行動並直接送出。

難度、死亡與收束

三種難度：標準歸零不會結束；困難歸零要擲生死檢定保命，NPC 只有一次復活；極限最難，NPC 死亡無法復活。

極限模式歸零會觸發「最後掙扎」，擲生死檢定通過就撐住，並拿到幾回合護盾。

「喘息」鈕可主動擲檢定回一點 HP／SAN，用來脫離死亡螺旋（回復上限為 HP／SAN 總量的 50%，不能連按）。

部分劇情道具有回復效果，在角色面板道具區按「使用」。

日常劇情也能回復：充分休息、過夜、接受治療或被安撫時，AI 會依事件小幅回復 HP／SAN（重複躺著刷不了血）。

名稱帶屬性標記的裝備道具（例如「幸運硬幣（智力+1）」）持有期間會自動加進對應骰點，失去道具即失效。

屬性點滿 20 後，該屬性按鈕會轉為鍛造：花 2 成長點兌換「成長結晶」（該屬性判定 +1，與裝備共用 +3 上限）；結晶是道具，可能因劇情失去。

成長與里程碑

達成重大成就會拿到成長點，可在「角色成長」區提升屬性或新增擅長領域，稱號隨成就升階。

達成情境的「本場目標」會觸發章節收束，由 AI 收尾並給成長點。

NPC 好感滿值、關係破裂或死亡等重大節點，系統會自動記進 Flags 與冒險日誌；好感跌到谷底（−100）會解鎖特殊成就。

面板上的像素愛心會隨好感改變動作：越愛越活潑，滿值會發光跳躍，觸底則整顆倒下。

冒險日誌記完整事件；想收藏對話用日記（泡泡右鍵、長按選「收藏這段」），在日記頁面對翻頁列桌機右鍵、手機長按可把整篇存成長圖。

讓 AI 不忘記、不跳針

故事變長後 AI 可能變鈍，這是模型限制。可以換標有 ▪ 的模型、多用「整理摘要／整理紀錄」壓縮記憶，或開「神」模式直接說明目前場景與重點。

存檔、備份與成人模式

每份遊戲紀錄綁定一份配置；另存新檔會複製紀錄並建立專屬配置。

匯出備份會保存紀錄、配置與頭像（不含 API Key），方便換裝置或還原。

成人模式在狀態面板開關（預設關），開啟後 AI 會在角色皆成年且合意的前提下更自然承接親密場景，實際尺度仍取決於所選模型。
